# Helixa Rebuild

This is the rebuild repository for Helixa — starting with glass UI and animated gradient background.

## Current Features
- Glass header with blur backdrop
- Animated gradient background shifting through shades of dark blue/purple
- Responsive glass cards
- Orbitron font for headers, Inter for body
- Helixa color palette (mint, lavender, blue, pink)

## Next Steps
1. Add interactive Trust Graph visualization
2. Connect to Helixa API endpoints
3. Implement wallet connection (Privy)
4. Add agent login (SIWA + x402)
5. Mobile responsiveness improvements

## Local Development
Open `index.html` in a browser to see the current UI.
